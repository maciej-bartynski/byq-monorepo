# Development #

1. mongo db
- open bash
- run mongo (ex. `mongod`)

2. api server
- open bash
- cd backend
- npm run dev

3. auth0 mock server
- open bash
- cd auth0.dev
- npm run dev

4. proxy server
- open bash
- cd frontend
- npm run dev

5. react app
- open bash
- cd frontend/react-app
- npm run start
